spec.policy.signedIdentity.matchPolicy <string> (required) default=""	# matchPolicy is a required filed specifies matching strategy to verify the image identity in the signature against the image scope. When set to "MatchRepoDigestOrExact", the identity in the signature must be in the same repository as the image identity if the image identity is referenced by a digest. More general scopes are prefixes of individual-image scopes, and specify a repository (by omitting the tag or digest), a repository namespace, or a registry host (by only specifying the host name and possibly a port number) or a wildcard expression starting with `*.`, for matching all subdomains (not including a port number). The policy requirements for more general scopes are ignored.
